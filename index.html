Mis servicios

1

Seo Marketing

SEO, acrónimo de Search Engine Optimization que en español es mejora de motores de búsqueda, son el conjunto de técnicas que se emplean para mejorar el posicionamiento (la visibilidad).

2

Diseño grafico

El diseño gráfico es una especialidad o profesión, cuyo objetivo es saciar las necesidades de comunicación visual con un fin específico, como, la publicidad. aquí se vale de herramientas para su ejecución,.

3

Marketing Digital

El marketing digital es la aplicación de las estrategias de comercialización llevadas a cabo en los medios digitales. Todas las técnicas del mundo off-line son imitadas y traducidas al mundo online.

4

Servidor web

El backend es la parte del desarrollo web que se encarga de que toda la lógica de una página web funcione. Se trata del conjunto de acciones que pasan en una web con la comunicación con el servidor.

5

Desarrollo Web

Desarrollo web significa edificar y conservar sitios web; es el trabajo que tiene sitio en un segundo plano y que posibilita que una web tenga una aspecto impecable, un desempeño veloz y un óptimo.

6

Diseño web

El diseño web involucra trabajo referente con el layout y diseño de páginas en línea, así como la producción de contenido, aunque principalmente se aplica a la construcción de sitios web.
